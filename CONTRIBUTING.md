# Contributing to aish.sh

We welcome marketing, documentation, and product contributions to aish!

## How to Contribute

1. Fork the repo
2. Create a feature branch (`git checkout -b feature/your-idea`)
3. Make your changes
4. Submit a pull request with a clear description

## Content Guidelines

- **Markdown**: Use clear, concise language
- **Links**: Keep internal links relative; external links absolute
- **Images**: Place in the same directory as the markdown file, named descriptively
- **Style**: Follow the [aish brand guidelines](./design/brand.md)

## Sections

- **docs/**: User tutorials and API reference
- **marketing/**: Positioning, messaging, case studies
- **product/**: Roadmap, proposals, competitive analysis
- **design/**: Brand guidelines and visual identity

Questions? Open an issue on [GitHub](https://github.com/LightHeart-Ventures/aish.sh).
