# aish.sh — Marketing & Product Strategy

This repository holds the **marketing, documentation, and product strategy** for [**aish**](https://github.com/LightHeart-Ventures/aish) — the AI-native shell.

aish is a no-bash shell built entirely from AI orchestration: instead of a POSIX shell, it's a stateful agent loop that understands intent, manages tools (MCP), reads/writes files, and runs programs. It's the foundation for building **AI-native workflows** on Linux.

## Contents

- **Marketing**: positioning, messaging, case studies, pricing strategy
- **Documentation**: user guides, tutorials, architecture deep-dives
- **Product**: roadmap, feature proposals, competitor analysis
- **Community**: contributor guidelines, engagement strategy
- **Design**: wireframes, style guides, brand identity
